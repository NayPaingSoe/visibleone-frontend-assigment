# Portfolio Website - Frontend Assignment

A modern, responsive portfolio website built with HTML and CSS, featuring a dark theme with vibrant gradient accents.

## Features

- **Responsive Design:** Fully responsive layout that adapts to mobile, tablet, and desktop screens.
- **Modern UI/UX:**
    - **Gradient Buttons & Borders:** Custom buttons with gradient borders and hover effects.
    - **Glassmorphism:** Header with backdrop blur effect.
    - **Grid Background:** Subtle grid background with decorative sparkles.
    - **Testimonials:** Stylish testimonial cards with fading gradient borders.
    - **Project Cards:** Featured projects section with numbered indicators.
- **Interactive Elements:**
    - Smooth hover animations for buttons and cards.
    - Mobile hamburger menu with custom styling.
    - "Hire me" CTA with gradient glow.

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- SVG Icons

## Setup & Usage

1.  Clone the repository.
2.  Open `index.html` in your browser or serve it using a local server (e.g., `python3 -m http.server`).
